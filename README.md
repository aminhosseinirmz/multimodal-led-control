\# Multimodal LED Control



Android + Arduino system to control RGB LEDs using:

\- Touch

\- Voice

\- Hand gestures



\## Features

\- Bluetooth LED control

\- Voice commands (Google Speech API)

\- Gesture recognition (MediaPipe + CameraX)

\- Jetpack Compose UI



\## Tech

\- Kotlin

\- Android Studio

\- Arduino Uno + HC-06



\## How it works

The Android app sends LED data via Bluetooth.

Arduino receives packets and updates LED colors using PWM.
